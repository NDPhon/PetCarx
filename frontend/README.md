
# PetCare Frontend

Hệ thống quản lý phòng khám thú y được xây dựng bằng React, Vite, và TypeScript.

## Yêu cầu
- Node.js (phiên bản 18 trở lên)

## Cài đặt và Chạy

Sau khi pull code về, bạn chỉ cần thực hiện 2 lệnh sau trong terminal:

1. **Cài đặt các thư viện cần thiết:**
   ```bash
   npm install
   ```

2. **Khởi động server phát triển:**
   ```bash
   npm run dev
   ```

Ứng dụng sẽ tự động chạy tại địa chỉ: [http://localhost:3000](http://localhost:3000)

## Cấu hình nổi bật
- **Port:** Đã được cấu hình cố định tại cổng `3000` trong file `vite.config.ts`.
- **CSS:** Sử dụng Tailwind CSS thông qua CDN (đã tích hợp trong `index.html`).
- **Icons:** Sử dụng thư viện `lucide-react`.
